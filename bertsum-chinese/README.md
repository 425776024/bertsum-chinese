# BERTSUM中文摘要抽取代码（魔改）

- 2.运行src/preprocess_LAI.py把json数据转成pt形式的二进制数据

- 3.运行src/train_LAI.py 开始训练

# 大致原理
-（不懂的加QQ/微信一定指导）
- 对文本句子进行分句（0/1），1是关键句，即，关键句分类。PS：文本长度超过512，切成多段低于512的。
- 文本有7句话：则是对7个维度的[CLS]位置向量输出0/1预测
- 抽取式摘要，效果咋不好说，有好有坏，还不错
